## What is Astro Utils?

<CardGrid stagger>
	<Card title="Web Forms" icon="pencil">
		Simple Web Forms Client & Server validation. Includes CSRF protection, JWT (JSON Web Tokens) based session and more.
	</Card>
	<Card title="Astro Hooks" icon="add-document">
		Client-Server hooks for Astro (onClick, onSubmit, etc)
        Allow you to easily create forms, without having to write any JavaScript.
	</Card>
	<Card title="Express Endpoint" icon="setting">
		Express like endpoint for Astro, with built-in body parsing.
        Support for multipart/form-data file uploads, express middleware and more.
	</Card>
	<Card title="Read the docs" icon="open-book">
        Easy to use, and easy to extend, Astro Utils is a great way to get started with Astro.

		Learn more in [Get started guild](guides/forms/).

	</Card>
</CardGrid>
